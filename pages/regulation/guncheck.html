Le guncheck

Le guncheck est un atout important que les gardiens peuvent utiliser pour maintenir le contrôle de la prison. Il permet de vérifier que les prisonniers n'ont pas d'armes et de calmer les situations tendues.

  • Déclenchement : Un guncheck peut être ordonné par le Warden dans les situations suivantes :

    • Lorsqu'un prisonnier est vu avec une arme à l'ordre (une arme en main alors qu'il ne devrait pas l'avoir).

    • Après la mort d'un gardien, pour rétablir le contrôle.

    • Si des coups de feu ont été tirés, qu'ils viennent d'un prisonnier ou non.

  • Limite de gunchecks par round : Un maximum de deux gunchecks peut être donné par round.

  • Durée : Un guncheck ne doit pas durer plus de 45 secondes, sauf si le Warden décide de l’interrompre avant la fin.

  • Guncheck avec position indiquée

    Lorsqu'un guncheck est déclenché avec une position spécifique, les prisonniers doivent respecter les instructions suivantes :

    • Immobile : Les prisonniers ne doivent pas bouger.

    • Face au repère : Ils doivent se tourner vers le point indiqué par le Warden (un mur, une zone spécifique).

    • Regardant leurs pieds : Ils doivent regarder leurs pieds pendant toute la durée du guncheck.

  • Guncheck sans position indiquée

    Si le Warden ne spécifie pas de position précise, les prisonniers doivent alors se conformer aux règles suivantes :

    • Dépacké : Les prisonniers doivent se disperser, ne pas rester en groupe.

    • Debout : Ils doivent rester debout.

    • Regardant le sol : Ils doivent baisser la tête et regarder le sol.

    • Immobile : Ils doivent rester immobiles.
