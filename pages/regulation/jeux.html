Les jeux

Les Gardiens doivent organiser des jeux pour les prisonniers. Ces jeux doivent être variés et offrir une chance équitable à tous les participants. Les jeux des maps doivent être privilégiés avant de recourir aux jeux hors-map comme "Jacques à Dit" ou "Réflexe", dans la mesure du possible, et en fonction du nombre de prisonniers évadés.

  • Jeux de la map priorisés : Les gardiens doivent d'abord utiliser les jeux proposés par la map. Les jeux hors-map sont autorisés, mais il faut varier les types de jeux et éviter la répétition constante des mêmes jeux.

  • Variété et créativité : Il est important de varier les jeux pour ne pas créer de monotonie. Les gardiens doivent faire preuve d'imagination pour diversifier les activités proposées aux prisonniers.

  • Jeu équitable : Tous les jeux doivent être équitables et permettre à chaque prisonnier de gagner à chances égales. Un jeu ne doit pas être conçu pour favoriser ou désavantager certains participants.

  • Élimination progressive : Il est interdit d’éliminer tous les prisonniers dès le premier jeu, sauf s'il reste moins de 4 prisonniers. Il faut privilégier les jeux qui offrent plusieurs étapes avant l'élimination finale.

  • Décompte obligatoire : Tout jeu commence par un décompte clair, notamment pour des jeux tels que la corde à sauter, le wipeout ou les courses.

  • Évasion : Un prisonnier qui s'échappe d'un jeu en cours est automatiquement considéré comme perdant et peut être éliminé par les gardiens.

  • Élimination dans les jeux individuels : Dans les jeux d’élimination individuelle (ex. : Questions/Réponses, Jacques à Dit, Disco), seul le chef des Gardiens (Warden) a le droit d’éliminer les prisonniers.

  • Format : Les questions doivent être posées de manière claire, soit par écrit dans le chat, soit à l'oral de manière audible pour tous les prisonniers.

  • Réponses : Les réponses doivent être données par écrit uniquement, et seule la réponse correcte est acceptée.

  • Niveau des questions : Les questions doivent être accessibles à une personne de niveau troisième (environ 13 ans ou plus), afin de rester justes et compréhensibles pour l'ensemble des joueurs.
